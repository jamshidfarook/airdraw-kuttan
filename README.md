# 🎨 KUTTAN · AirDraw ULTRA

AI-powered hand gesture drawing app — draw in the air using your hand.

---

## 🚀 Live Demo  
https://jamshidfarook.github.io/airdraw-kuttan/

---

## ✨ Features
- Hand tracking drawing (MediaPipe)
- Gesture controls (draw, color, size, mode)
- Multiple brush modes
- Dynamic backgrounds
- Save drawings as PNG

---

## 🎮 Controls
- Draw → Index finger  
- Color → Index + Middle  
- Size → 3 fingers  
- Mode → 4 fingers  
- Clear → Hold fist  
- Save → Pinky  

---

## 🧠 Tech
- HTML, CSS, JavaScript  
- MediaPipe  
- Canvas API
